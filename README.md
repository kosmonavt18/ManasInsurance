# ManasInsurance

## О проекте
ManasInsurance — учебное веб‑приложение страховой компании на Django. Проект демонстрирует
каталог страховых продуктов, калькуляторы тарифов, прием заявок и базовые страницы сайта
(главная, о компании, контакты, страховые случаи).

## Основные возможности
- Каталог страховых продуктов для частных, корпоративных клиентов и автострахования.
- Отдельные страницы продуктов с описаниями, ценой «от» и изображениями.
- Таблицы тарифов для авто‑продуктов (ОСАГО, КАСКО, ДСАГО, НС).
- Калькуляторы стоимости (ОСАГО, КАСКО, имущество, путешествия).
- Прием заявок с выбором типа страхования и загрузкой техпаспорта для авто.
- Пользовательские аккаунты с ролями (админ/сотрудник/клиент).
- Страницы «Страховые случаи» и «Контакты» с филиалами и инструкциями.
- Мультиязычность (ru/ky) и админ‑панель для управления контентом.

## Что внутри (структура приложений)
- `apps/core` — базовые страницы, настройки сайта, филиалы и страховые случаи.
- `apps/accounts` — расширенная модель пользователя и полисы клиентов.
- `apps/clients` — продукты для частных клиентов (жизнь, здоровье, путешествия, имущество).
- `apps/corporate` — продукты для бизнеса (имущество, ДМС, банки, ответственность).
- `apps/auto` — авто‑продукты и таблицы тарифов.
- `apps/applications` — заявки клиентов и их статусы.
- `apps/calculators` — сервисы и формы калькуляторов.

## Технологии
- Django 5.x
- SQLite (по умолчанию)
- Pillow (загрузка изображений)
- Django Templates, i18n (ru/ky)

## Быстрый старт
1) Установите зависимости:
```
pip install -r requirements.txt
```

2) Примените миграции и создайте суперпользователя:
```
python manage.py migrate
python manage.py createsuperuser
```

3) (Опционально) Загрузите демоданные:
```
python manage.py loaddata fixtures/demo.json
```

4) Запустите сервер:
```
python manage.py runserver
```
