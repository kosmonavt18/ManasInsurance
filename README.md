# ManasInsurance

Веб-приложение на Django для страхового агентства. Проект содержит каталог
страховых продуктов, онлайн-заявки, личный кабинет клиента и калькуляторы
тарифов для популярных видов страхования в Кыргызстане.

## Возможности

- Главная страница с подборками страховых продуктов.
- Разделы страхования для частных клиентов, корпоративных клиентов и авто.
- Детальные страницы продуктов с описанием и тарифами.
- Калькуляторы: ОСАГО, КАСКО, имущество, путешествия.
- Онлайн-заявки с загрузкой техпаспорта для автострахования.
- Личный кабинет: заявки и полисы пользователя.
- Мультиязычность (ru/ky), админ-панель Django.

## Технологии

- Python + Django
- SQLite (по умолчанию)
- Pillow для работы с изображениями

## Быстрый старт

1. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Примените миграции:
   ```bash
   python manage.py migrate
   ```
4. (Опционально) загрузите демо-данные (если доступен файл `fixtures/demo.json`):
   ```bash
   python manage.py loaddata fixtures/demo.json
   ```
5. (Опционально) создайте суперпользователя:
   ```bash
   python manage.py createsuperuser
   ```
6. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

После запуска откройте:
- приложение: http://127.0.0.1:8000/
- админ-панель: http://127.0.0.1:8000/admin/

## Продакшен

- Установите `SECRET_KEY` через переменную окружения и отключите `DEBUG`.
- Заполните `ALLOWED_HOSTS` актуальными доменами.
- Настройте `DATABASES` под нужную СУБД (PostgreSQL/MySQL и т.д.).
- Выполните `python manage.py collectstatic` и раздавайте статику/медиа через веб-сервер.

## Полезные заметки

- База данных по умолчанию — `db.sqlite3` в корне проекта.
- Медиа-файлы сохраняются в папку `media/`.
