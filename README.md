# ManasInsurance

Профессиональный веб‑проект страхового агентства на Django. Сайт включает публичные страницы, каталог продуктов, калькуляторы, личный кабинет, отправку заявок (в том числе автострахование с загрузкой фото техпаспорта), админ‑панель и мультиязычность (RU/KY).

## Возможности

- Публичные разделы: главная, о компании, контакты, страховые случаи.
- Продукты для частных и корпоративных клиентов.
- Автострахование (ОСАГО/КАСКО/ДСАГО) и сопутствующие услуги.
- Заявки от клиентов с загрузкой изображения техпаспорта.
- Админ‑панель с фильтрами, поиском и предпросмотром файлов.
- Калькуляторы страхования.
- Мультиязычность (i18n).

## Технологии

- Python 3.11+
- Django 5.x
- SQLite (по умолчанию)
- HTML/CSS/JS, шаблоны Django

## Быстрый старт

### 1) Установка зависимостей

```powershell
python -m pip install -r requirements.txt
```

### 2) Миграции

```powershell
python manage.py migrate
```

### 3) Создание суперпользователя

```powershell
python manage.py createsuperuser
```

### 4) Запуск сервера разработки

```powershell
python manage.py runserver
```

Откройте: `http://127.0.0.1:8000/`

Админ‑панель: `http://127.0.0.1:8000/admin/`

## Загрузка демо‑данных (опционально)

Если хотите заполнить сайт тестовыми данными:

```powershell
python manage.py loaddata fixtures/demo.json
```

## Конфигурация

- Основные настройки находятся в `settings.py`.
- Медиа‑файлы (загруженные изображения) сохраняются в `media/`.
- Статика находится в `static/`.

## Структура проекта

- `apps/` — приложения Django (accounts, applications, auto, calculators, clients, core, corporate).
- `templates/` — HTML‑шаблоны.
- `static/` — CSS/JS/изображения.
- `media/` — пользовательские загрузки.

## Полезные команды

Проверка проекта:

```powershell
python manage.py check
```

Сборка статики (если потребуется для продакшена):

```powershell
python manage.py collectstatic
```

## Примечания

- Файлы базы данных и медиа исключены из git через `.gitignore`.
- Для корректной загрузки файлов в заявке форма использует `enctype="multipart/form-data"`.
